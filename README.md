## How tu run it
It's standard Spring Boot application. You will need ***Java 8***.

1. Start server

    Windows
    ```
    mvnw spring-boot:run
    ```
    Linux
    ```
    ./mvnw spring-boot:run
    ```
    
    or see https://docs.spring.io/spring-boot/docs/current/reference/html/using-boot-running-your-application.html for more options

2. Open web browser and type http://localhost:8080
